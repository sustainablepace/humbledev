Text:

In order to get the Maven configuration right, I wanted to have a local (link:http://www.sonarqube.org/ text:Sonarqube) to test. With (link:https://www.docker.com/ text:Docker), this is totally trivial.

## Run the Docker container ##

You should already have Docker running on your local machine. Download the Sonarqube container from Docker Hub like this

```
docker pull sonarqube
```

After downloading, start the container via

```
docker run -d --name sonarqube -p 9000:9000 -p 9092:9092 sonarqube:5.1
```

You should see a Sonarqube installation at

```
http://localhost:9000
```

## Run Maven goal ##

Simply run the goal with the local installation

```
mvn sonar:sonar -Dsonar.host.url=http://localhost:9000
```

You should see the generated metrics at

```
http://localhost:9000
```

P.S. Read more about the Sonarqube container at (link:https://hub.docker.com/_/sonarqube/ text:Docker Hub).
